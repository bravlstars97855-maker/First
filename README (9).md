## Описание

`GitHub User Finder` — это десктопное GUI-приложение для поиска пользователей на GitHub по их логину. Приложение позволяет просматривать базовую информацию о пользователе и вести список "избранных" профилей.

### Основные возможности:

*   **Поиск пользователя:** Ввод логина для поиска через официальный GitHub API.
*   **Просмотр информации:** Отображение имени, логина, числа подписчиков/подписок и биографии.
*   **Кликабельная ссылка:** Логин пользователя является ссылкой на его профиль, которая открывается в браузере.
*   **Список избранного:** Возможность добавлять пользователей в локальный список избранного.
*   **Сохранение избранного:** Список избранных пользователей сохраняется в файл `favorites.json` и загружается при старте.

## Технологии

*   **Язык:** Python 3
*   **GUI:** `tkinter`
*   **HTTP-запросы:** `requests`
*   **Формат данных:** JSON
*   **Система контроля версий:** Git

## Как использовать API

Приложение использует публичный и бесплатный REST API GitHub. Для получения информации о пользователе отправляется GET-запрос на эндпоинт:

```
https://api.github.com/users/<USERNAME>
```

Где `<USERNAME>` — это логин искомого пользователя.

**Ограничения:** Публичный API имеет лимит на количество запросов от одного IP-адреса (обычно около 60 запросов в час). Для данного приложения этого более чем достаточно. При превышении лимита API вернет ошибку с кодом `403`.

## Примеры использования (Как запустить)

1.  **Клонируйте репозиторий:**
    ```bash
    git clone <URL вашего репозитория>
    cd <папка репозитория>
    ```

2.  **Установите зависимости:**
    Приложению требуется библиотека `requests`. Установите ее с помощью pip:
    ```bash
    pip install requests
    ```

3.  **Запустите приложение:**
    ```bash
    python github_finder.py
    ```

4.  **Работа с интерфейсом:**
    *   **Поиск:** Введите логин пользователя (например, `torvalds` или `google`) в поле ввода и нажмите «Найти» или клавишу Enter.
    *   **Добавление в избранное:** После успешного поиска нажмите кнопку «Добавить в избранное».
    *   **Удаление из избранного:** Выберите пользователя в списке «Избранное» и нажмите кнопку «Удалить из избранного».
