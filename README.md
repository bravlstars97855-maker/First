# GitHub User Finder

**Автор:** Давид Кабанин

## Описание
Это простое GUI-приложение для поиска пользователей GitHub с возможностью добавления в избранное. Он использует API GitHub для получения информации о пользователях и сохраняет избранных в локальный файл JSON.

## Как использовать API
1. Введите логин пользователя в поле поиска.
2. Нажмите кнопку «Поиск» для получения информации.
3. Для добавления пользователя в избранное выберите его из результатов и нажмите «Добавить в избранное».

## Примеры использования
- Поиск пользователя: `torvalds`
- Добавление в избранное: выбранный пользователь из списка.

## Важные замечания
- Поле поиска не должно быть пустым.
- Все избранные пользователи сохраняются в файле `favorites.json`.
